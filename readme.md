

# lexxtract general law


[live website](https://dkormann.github.io/lexXtract-general-law)

agentic data extraction and organisation tool


## development

### frontend

the frontend is a Typescript project

install using the bun package manager.

- installation:
  `bun install`
- start dev server:
  `bun run dev`
- build for deployment:
  `bun run buid`
  will build static website into ./docs folder

### backend

backend is in the database folder. It is a spacetimedb project.
